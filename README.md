# API Harga Emas Indonesia

API ini menyediakan data harga emas dari berbagai brand di Indonesia, termasuk:

- **GALERI 24**
- **EMASKU** / **EMASKITA**
- **ANTAM**
- **UBS**
- **WARIS SAMPOERNA**

Dan : 
- **DINAR G24**
- **BABY GALERI 24**
- **ANTAM MULIA RETRO**
- **ANTAM NON PEGADAIAN**
- **LOTUS ARCHI**
- **UBS DISNEY**
- **UBS ELSA**
- **UBS ANNA**
- **UBS MICKEY FULLBODY**
- **LOTUS ARCHI GIFT**
- **UBS HELLO KITTY**
- **BABY SERIES TUMBUHAN**
- **BABY SERIES INVESTASI**
- **BATIK SERIES**

Data diperbarui di jam-jam tertentu, diambil langsung dari website [Galeri 24](https://galeri24.co.id/) ,[Antam](https://logammulia.com/) , [Waris Sampoerna](https://sampoernagold.com/) , [Emasku / Hartadinata](https://hartadinataabadi.co.id/) , [Pegadaian](https://pegadaian.co.id/).

## 🚀 Cara Penggunaan
Halaman utama dapat dilihat disini [Main Page](https://emas.maulanar.my.id)

Cara penggunaan dapat dilihat di [Docs](https://emas.maulanar.my.id/api/docs)

## 🎯 Manfaat API Ini

- Mendapatkan informasi harga emas dari berbagai brand dengan mudah.
- Data selalu diperbarui otomatis.
- Dapat digunakan untuk aplikasi investasi emas atau e-commerce.
